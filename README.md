# Ayush Bot 🤖

A simple chatbot made using Kotlin that runs in the command line.

## Features

* Greets the user
* Takes name input
* Guesses age
* Counts numbers
* Asks a simple question

## How to Run

1. Clone the repo
2. Open in IntelliJ
3. Run main()
